Basic Controls

All key bindings can be remapped in the game settings. The defaults listed here are the standard layout.

General

Action Default Key
Chat Enter
Select / look at entity F
Open inventory I
Open navigation panel N
Open faction menu H

Astronaut Mode — With Gravity

Action Default Key
Move forward W
Move backward S
Strafe left A
Strafe right D
Jump Space
Walk (slow) Left Shift
Look around Mouse
Activate / interact R
Select item in hotbar 1–9, 0 / Mouse wheel
Use / fire selected item Left Mouse Button
Secondary use Right Mouse Button

Astronaut Mode — Zero Gravity

In zero-G all the same movement keys apply, plus:

Action Default Key
Move up Space
Move down Left Ctrl
Align to nearby structure Space (tap while near)

Tip: Aligning to a structure also causes you to move with it if the structure is flying. This is useful for boarding moving ships.

Toggle build / flight mode Z
