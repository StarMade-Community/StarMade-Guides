# Ship Basics

A ship is the primary way to travel through space, fight enemies, and carry cargo. Every ship is built around a **Ship Core** block — the single block that defines the ship.

## Entering and Exiting a Ship

| Action | Key |
|--------|-----|
| Enter a ship (look at core) | `R` |
| Exit a ship (look away from ship) | `R` |

When you enter a ship you are placed directly into its **Ship Core**. From there you can switch between the two ship modes.

## Ship Modes

Press `Z` to toggle between flight mode and build mode while inside a ship.

## Ship Status Bars

Every ship has three resource bars visible in the HUD:

| Bar | Colour | Description |
|-----|--------|-------------|
| **Hull / HP** | Red | Total hull health. Reaches zero → ship destroyed |
| **Shield** | Blue | Absorbs incoming damage before hull is hit |
| **Power** | Green | Energy available for weapons, thrusters, and shields |

## Hull and Armour

Blocks making up the ship's hull each have individual hit points. The **total ship HP** is the sum of all block HP.

- **Hull blocks** provide high HP per block.
- **Armour blocks** reduce damage from each incoming hit before it is applied to HP — they act as a damage multiplier reduction.
- Weapons, computers, and other functional blocks are more fragile than hull/armour.

A good ship design protects critical systems (reactors, computers, the core) with layers of hull and armour blocks.

## Rebooting

If your power runs completely out while under attack, the ship enters a **reboot** state. Systems come offline temporarily. Keeping spare power capacity or using shields to buy time prevents this.

## Ship Controls (Flight Mode)

| Action | Key |
|--------|-----|
| Accelerate forward | `W` |
| Decelerate / reverse | `S` |
| Strafe up | `Space` |
| Strafe down | `Left Ctrl` |
| Strafe left / right | `A` / `D` |
| Roll left / right | `Q` / `E` |
| Fire active weapon | `Left Mouse Button` |
| Cycle weapon slots | `1`–`9`, `0` / Mouse wheel |
| Select target | `F` |
| Open weapon panel | `G` |
| Open structure panel | `Delete` |
